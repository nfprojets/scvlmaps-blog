SC-VLMaps: Depth-Free Visual–Language Mapping via Scene Coordinate Regression

Nanda Febri Istighfarin1, Baehoon Choi2, HyungGi Jo1
1Jeonbuk National University, 2Bstar Robotics Co., Ltd.
Paper Code

We introduce SC-VLMaps, a depth-free visual–language mapping framework that constructs semantic map using only monocular RGB input by leveraging scene coordinate regression network.

Abstract

The ability to connect visual observations with human language is increasingly valuable for embodied agents in tasks such as navigation and semantic mapping. Existing visual–language map (VLMaps) approach enables this connection but typically depends on depth images to project semantic features into 3D space, which limits scalability due to sensor cost and deployment constraints. In this work, we introduce SC-VLMaps, a depth-free visual–language mapping framework that constructs semantic maps using only monocular RGB input. SC-VLMaps leverages a scene coordinate regression (SCR) network to predict dense 3D coordinates from images, bypassing the need for depth supervision and enabling implicit geometry reconstruction. The predicted coordinates are fused into a voxel grid and augmented with language-aligned features from a frozen visual–language encoder, producing maps that are both geometrically coherent and semantically enriched. By employing a multi-scene training strategy, SC-VLMaps generalizes from indoor datasets (7Scenes) to challenging outdoor benchmarks (Cambridge Landmarks). Experiments show that SC-VLMaps achieves denser, more compact maps with stronger semantic alignment than VLMaps, while requiring only monocular RGB images.

VLMaps: Our Framework

SC-VLMaps operates in two stages: offline and online. (a) Offline stage: We train a scene coordinate regression network with a shared encoder and scene-specific heads to predict dense 3D scene coordinates. (b) Online stage: The predicted coordinates are fused with language-aligned embeddings to generate a top-down map that is semantically indexed for language-based queries.


Experiments

We compare SC-VLMaps with the baseline VLMaps using two metrics: semantic alignment score (SAS) and voxel occupancy. The evaluation is performed on the indoor 7Scenes dataset and further extended to the outdoor Cambridge Landmarks dataset, where the baseline method fails to generate maps. Across all experiments, SC-VLMaps achieves better performance than the baseline.
